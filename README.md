# Checkout com Cartão de Crédito

Este projeto foi desenvolvido como parte do desafio Daily UI, com foco na criação de uma interface de checkout profissional, limpa e intuitiva.

## Objetivo
Criar uma página de pagamento com cartão de crédito contendo os principais elementos de um checkout real, como:
- nome no cartão
- número do cartão
- validade
- CVV
- resumo do pedido
- confirmação de pagamento

## Tecnologias utilizadas
- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
- JavaScript

## Funcionalidades
- formulário de pagamento estilizado
- máscara para número do cartão
- máscara para validade
- validação básica dos campos
- modal de confirmação ao finalizar o pagamento

## Observação
Projeto criado para prática de interface, layout e experiência do usuário.
